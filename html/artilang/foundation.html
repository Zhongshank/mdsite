人造語協會

人造語基金會目前沒有《基金》，所以比較像是《協會》！

以下是我對《人造語基金會》的成立想法。

  • https://www.facebook.com/ccckmit/posts/10155003282756893
我終於想清楚該怎樣推廣《人造語》，還有自己的角色定位了！

我發現成立公司是不恰當的，成立《人造語基金會》才是我應該做的事情。

我目前的在《人造語》應該類似 Linus Torvalds 在 Linux 作業系統上的扮演的角色一樣，是程式核心的維護者。

我釋出的人造語相關程式碼，還有字典，將會是《人造語》的第一批資源，《人造語基金會》的目的是確保這些《開源資產》能夠被長期維護並持續發展更新，另外還要維護《人造語》的中立性，避免被某些公司佔有成為私人資產。

Linus Torvalds 本人沒從 Linux 上賺很多錢，但是卻讓整個開源軟體產業蓬勃發展。

我想我應該也不會從《人造語》上賺很多錢，但是卻可以扮演《人造語產業》發展背後的推手角色！

《人造語基金會》將持續釋出的《程式、字典、論文、書籍》。

這些資源將會採用類似 Linux 的《開源授權》。

《人造語核心程式 kernel》將採用 GPL 授權，任何對 kernel 的修改都必須開放原始碼，但是透過 API 呼叫 kernel 時將不受 GPL 的開放原始碼限制。

《人造語字典資源》將採用 CC:BY-SA 的《創作共用-姓名標示---相同方式分享授權》，任何由此資源衍伸出來的字典也必須採用 CC:BY-SA 的方式分享，這樣才能確保人造語資源不會成為商業公司的私有秘密資產。

基金會歡迎大家使用《人造語技術》去開商業公司，只要遵守上述授權即可。像是製作《人造語的手機 APP》，或者販賣內建人造語技術的《自動翻譯機》等等。

另外、為了避免商業公司《申請人造語的子專利》之後，用專利阻礙其他公司製造人造語產品，我們或許會考慮先申請《人造語的母專利》，這樣當某公司用子專利告其他公司時，我們就可以提供母專利給被告的公司當武器，避免因製造人造語產品而被控告或索取專利授權金。
