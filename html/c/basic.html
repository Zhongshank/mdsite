基礎語法

hello.c

檔案： hello.c

#include <stdio.h>

int main() {
  printf("hello 你好！\n");
}

執行結果

D:\code>gcc hello.c -o hello
D:\code>hello
hello 你好！

注意： 在 windows 8 當中我必須將 hello.c 儲存成 ansi 編碼，中文字才能正確印出。

範例 -- 加總 (sum.c)

使用 while 迴圈

檔案： wsum.c

#include <stdio.h>

int main() {
  int i=1, sum=0;
  while (i<=10) {
    sum = sum + i;
    printf("i=%d sum=%d\n", i, sum);
    i = i + 1;
  }
}

執行結果：

D:\Dropbox\cccwd\db\c\code>gcc wsum.c -o wsum

D:\Dropbox\cccwd\db\c\code>wsum
i=1 sum=1
i=2 sum=3
i=3 sum=6
i=4 sum=10
i=5 sum=15
i=6 sum=21

i=8 sum=36

i=10 sum=55

使用 for 迴圈

檔案：sum.c

#include <stdio.h>

int main() {
  int i, sum=0;
  for (i=1;i<=10;i++) {
    sum = sum + i;
    printf("i=%d sum=%d\n", i, sum);
  }    
}
