#範例 -- 加總 (sum.c)

###使用 while 迴圈

檔案： wsum.c

#include <stdio.h>
int main() {
  int i=1, sum=0;
  while (i<=10) {
    sum = sum + i;
    printf("i=%d sum=%d\n", i, sum);
    i = i + 1;
  }
}

執行結果：

###使用 for 迴圈

檔案：sum.c

#include <stdio.h>


int main() {
  int i, sum=0;
  for (i=1;i<=10;i++) {
    sum = sum + i;
    printf("i=%d sum=%d\n", i, sum);
  }    
}
