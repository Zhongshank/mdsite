再論估計與檢定

在上一章當中，我們說明了如何計算單一組樣本「平均值」的「信賴區間」與「檢定」等問題。在這一章當中，我們將進一步探索 如何用 R 軟體進行其他的檢定，像是「變異數」的檢定，兩組樣本的檢定等問題，我們將不會花太多力氣在數學說明上，而是採用 直接操作的範例導向方法，用範例說明這些檢定的實作方法。

檢定的分類

大致上來說，檢定可以分為「單組樣本、兩組樣本或多組樣本」的檢定，在上期當中，我們所檢定的對象是單組樣本的平均值 μ\muμ， 當然我們也可以檢定兩組樣本的平均值是否相等，也就是 μ1=μ2\mu_1=\mu_2μ1​=μ2​ 是否為真。

如果想檢定的關係並非「等號」關係，那麼檢定還可以分為「右尾檢定、左尾檢定、雙尾檢定」等三種，以平均值 μ\muμ 的檢定而言， 其檢定假設如下表所示：

\ 右尾檢定 左尾檢定 雙尾檢定
H0 μ≤μ0\mu \le \mu_0μ≤μ0​ μ≥μ0\mu \ge \mu_0μ≥μ0​ μ=μ0\mu=\mu_0μ=μ0​
H1 μ>μ0\mu \gt \mu_0μ>μ0​ μ<μ0\mu \lt \mu_0μ<μ0​ μ≠μ0\mu \neq \mu_0μ≠μ0​

另外、檢定的對象如果不是平均數 μ\muμ ，而是變異數 σ\sigmaσ ，或者某個比例 p，或者是中位數 M ，那就得改用對應的分布進行檢定，讓我們將檢定的種類與方法整理一下，列表如下：

  • 單組樣本的檢定

    • 平均值的檢定？ ( H0:μ=μ0H0: \mu = \mu_0H0:μ=μ0​ ) -- 學生 T 檢定
    • 變異數的檢定？ ( H0:σ=σ0H0: \sigma = \sigma_0H0:σ=σ0​ ) -- 卡方 χ2\chi^2χ2 檢定
      • 檢定式： (n−1)s2/σ02∼χ2(n-1) s^2/\sigma_0^2 \sim \chi^2(n−1)s2/σ02​∼χ2
    • 比例 p 的檢定？ ( H0:p=p0H0:p = p_0H0:p=p0​ ) -- 常態 Z 檢定
      • 檢定量： X/n=p^X/n = \hat{p}X/n=p^​
      • 檢定式： p^−pp(1−p)/n∼Z\frac{\hat{p}-p}{\sqrt{p(1-p)/n}} \sim Zp(1−p)/n​p^​−p​∼Z
    • 中位數 M 的檢定？ ( H0:M=M0H0: M = M_0H0:M=M0​ ) -- 無母數方法 Wilcoxon Sign-Rank 檢定

  • 兩組樣本的檢定

    • 比較兩平均數 μ1,μ2\mu_1, \mu_2μ1​,μ2​ 的差值 ( H0:μ1−μ2=0H0:\mu_1-\mu_2=0H0:μ1​−μ2​=0 ) 的檢定

      • 檢定式：(X1¯−X2¯)−(μ1−μ2)Sp2(1/n1+1/n2)∼Tn1+n2−2\frac{(\bar{X_1}-\bar{X_2})-(\mu_1-\mu_2)}{\sqrt{S_p^2 (1/n_1 + 1/n2)}} \sim T{n_1+n_2-2}Sp2​(1/n1​+1/n2​)​(X1​¯​−X2​¯​)−(μ1​−μ2​)​∼Tn1​+n2​−2​

    • 比較兩變異數 σ1,σ2\sigma_1, \sigma_2σ1​,σ2​ 的比例 ( H0:σ1=σ2H0:\sigma_1=\sigma_2H0:σ1​=σ2​ ) 的檢定

兩組樣本的平均值檢定

(方法一)：合併 T 檢定 (pooled T test) -- 又被稱為：「獨立 T 檢定」或「不相關 T 檢定」

  • 前提條件：兩組樣本必須互相獨立才能使用合併 T 檢定，沒有理相信 (常態分布) 兩組樣本的母體變異數不相等的情況之下，想比較 μ1−μ2\mu_1 - \mu_2μ1​−μ2​ 時，可用「合併 T 檢定」。

合併 T 檢定的數學原理，式考慮下列兩組分佈公式：

Z=(X¯−Y¯)−(μ1−μ2)σ2(1n1+1n2)∼N(0,1)Z=\frac{(\bar{X}-\bar{Y})-(\mu_1 - \mu_2)}{\sqrt{\sigma^2 (\frac{1}{n_1} + \frac{1}{n_2})}} \sim N(0,1)Z=σ2(n1​1​+n2​1​)​(X¯−Y¯)−(μ1​−μ2​)​∼N(0,1)

(n1−1)S12σ2∼χ(n1−1);(n2−1)S12σ2∼χ(n2−1)\frac{(n_1-1) S_1^2}{\sigma^2} \sim \chi (n_1-1) \;\;;\;\; \frac{(n_2-1) S_1^2}{\sigma^2} \sim \chi (n_2-1)σ2(n1​−1)S12​​∼χ(n1​−1);σ2(n2​−1)S12​​∼χ(n2​−1)

當 S1S_1S1​ 與 S2S_2S2​ 相互獨立時，會有下列性質：

(n1−1)S12σ2+(n2−1)S12σ2∼χ2(n1+n2−2)\frac{(n_1-1) S_1^2}{\sigma^2} + \frac{(n_2-1) S_1^2}{\sigma^2} \sim \chi^2(n_1+n_2-2)σ2(n1​−1)S12​​+σ2(n2​−1)S12​​∼χ2(n1​+n2​−2)

於是可以得到：

T=(X¯−Y¯)−(μ1−μ2)S2(1n1+1n2)∼t(n1+n2−2)T = \frac{(\bar{X}-\bar{Y})-(\mu_1 - \mu_2)}{\sqrt{S^2 (\frac{1}{n_1} + \frac{1}{n_2}})} \sim t(n_1+n_2-2)T=S2(n1​1​+n2​1​​)(X¯−Y¯)−(μ1​−μ2​)​∼t(n1​+n2​−2)

也就是 (X¯−Y¯)−(μ1−μ2)S2(1n1+1n2)\frac{(\bar{X}-\bar{Y})-(\mu_1 - \mu_2)}{\sqrt{S^2 (\frac{1}{n_1} + \frac{1}{n_2}})}S2(n1​1​+n2​1​​)(X¯−Y¯)−(μ1​−μ2​)​ 服從自由度為 (n1+n2−2)(n_1+n_2-2)(n1​+n2​−2) 的 T 分布。

其中的 S 定義如下：

S=(n1−1)S12+(n2−1)S22(n1−1)+(n2−1)S = \frac{(n_1-1) S_1^2 + (n_2-1) S_2^2}{(n_1 - 1) + (n_2-1)}S=(n1​−1)+(n2​−1)(n1​−1)S12​+(n2​−1)S22​​

變異數服從甚麼分布？

兩組樣本的變異數檢定

\ 右尾檢定 左尾檢定 雙尾檢定
H0 σ1=σ2\sigma_1 = \sigma_2σ1​=σ2​ σ1=σ2\sigma_1 = \sigma_2σ1​=σ2​ σ1=σ2\sigma_1 = \sigma_2σ1​=σ2​
H1 σ1>σ2\sigma_1 \gt \sigma_2σ1​>σ2​ σ1<σ2\sigma_1 \lt \sigma_2σ1​<σ2​ σ1≠σ2\sigma_1 \neq \sigma_2σ1​≠σ2​

那麼、對於變異數而言應該怎麼檢定才好呢？

由於單一樣本群的變異數服從卡方分布，因此在兩組樣本群時，會服從以下的分布：

χγ12/γ1χγ22/γ2\frac{\chi^2_{\gamma_1} / \gamma1}{\chi^2{\gamma_2} / \gamma_2}χγ2​2​/γ2​χγ1​2​/γ1​​

這種由兩個卡方分布相除的分布稱為 F 分布。

定理：

令 X, Y 分別是來自常態分配 N(μX,σX2)N(\mu_X, \sigma_X^2)N(μX​,σX2​) 與 N(μY,σY2)N(\mu_Y, \sigma_Y^2)N(μY​,σY2​) 的樣本群，SX2S_X^2SX2​ 與 SY2S_Y^2SY2​ 為 X , Y 兩群樣本的變異數，若 σX=σY\sigma_X=\sigma_YσX​=σY​ ，則統計量 SX2/SY2S_X^2 / S_Y^2SX2​/SY2​ 服從自由度為 nX,nYn_X, n_YnX​,nY​ 的 F 分布。

在 R 函數中，進行「兩組樣本變異數檢定」的函數是 var.test()，該函數即是採用 F 分布去進行檢定的，以下是一個用 var.test() 進行檢定的範例。

(方法二)：Wilcoxon Signed-Rank 檢定：兩組成對觀察值 (X, Y) 適用

  • 樣本：兩組成對觀察值 (X1,Y1),(X2,Y2),...,(Xn,Yn)(X_1, Y_1), (X_2, Y_2), ..., (X_n, Y_n)(X1​,Y1​),(X2​,Y2​),...,(Xn​,Yn​)
  • 方法：將差距絕對值 ∣X1−Y1∣,∣X2,Y2∣,...,∣Xn,Yn∣|X_1-Y_1|, |X_2, Y_2|, ..., |X_n, Y_n|∣X1​−Y1​∣,∣X2​,Y2​∣,...,∣Xn​,Yn​∣ 由小到大排序，並給予 1..n 的名次。
  • 檢定統計量：看看 MXM_XMX​ 是否夠接近 MYM_YMY​，如果差很多那麼 W 應該會很大。
  • 正排名權重：W+=∑Ri>0RiW+ = \sum{R_i>0} R_iW+​=∑Ri​>0​Ri​
  • 負排名權重：∣W−∣=∑Ri<0∣Ri∣;|W-| = \sum{R_i<0} |R_i| ;∣W−​∣=∑Ri​<0​∣Ri​∣;
  • W = min(W+, |W-|)

結語

在本文中，我們討論了各種「單組樣本」與「兩組樣本」檢定方法，並用 R 軟體進行示範操作，希望透過這樣的方式讓 讀者感受到「檢定」這一個統計工具的用途。

但是、我們還漏掉了一個部分，那就是「三組以上的樣本」之檢定方法，用來檢定這些樣本是否來自同一母體，這種 「多組樣本檢定」的問題，通常必須使用「變異數分析」(ANOVA, Analysis Of Variance) 的方法處理，這將是我們 下一章所要討論的主題！
