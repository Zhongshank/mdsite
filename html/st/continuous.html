均勻分布 (Uniform distribution)

意義：在範圍 (a,b) 內的出現機會均等。

R 函數：unif(a:min, b:max)

課本公式：f(x)=1b−af(x) = \frac{1}{b-a}f(x)=b−a1​

R 的公式：f(x) = 1/(max-min)

變數意義：a:min 範圍下限, b:max 上限

R 程式範例

> op=par(mfrow=c(2,2))
> curve(dunif(x, 0, 1), -2, 10)
> curve(dunif(x, 1, 5), -2, 10)
> curve(dunif(x, -1, 9), -2, 10)
> curve(dunif(x, 10, 110), 0, 200)
>

常態分布 (Normal Distribution)

意義：有誤差的對稱性分布形式，中央高兩邊低的形式。

繪圖：用 R 繪製標準差為 1, 2, 3, 的常態分布。

curve(dnorm(x), -5, 5, col="black")
curve(dnorm(x, sd=2), -5, 5, col="blue", add=T)
curve(dnorm(x, sd=3), -5, 5, col="green", add=T)

R 函數：norm(mean, sd)

公式：12πσe−12[(x−μ)/σ]2\frac{1}{\sqrt{2\pi} \sigma} e^{- \frac{1}{2} [(x-\mu)/\sigma]^2}2π​σ1​e−21​[(x−μ)/σ]2

R 的公式：f(x) = 1/(√(2 π) σ) e^-((x - μ)^2/(2 σ^2))

重要性：根據中央極限定理，任何 n 個獨立樣本的平均值趨近於常態分布。

中央極限定理： x1+x2+...+xn→N(μ,δ/n)x_1+x_2+...+x_n \rightarrow N(\mu, \delta/\sqrt{n})x1​+x2​+...+xn​→N(μ,δ/n​)

標準差

1 標準差： P[−1σ<X−μ<1σ]≃0.68P[-1 \sigma < X-\mu < 1 \sigma] \simeq 0.68P[−1σ<X−μ<1σ]≃0.68 .

2 標準差： P[−2σ<X−μ<2σ]≃0.95P[-2 \sigma < X-\mu < 2 \sigma] \simeq 0.95P[−2σ<X−μ<2σ]≃0.95 .

3 標準差： P[−3σ<X−μ<3σ]≃0.997P[-3 \sigma < X-\mu < 3 \sigma] \simeq 0.997P[−3σ<X−μ<3σ]≃0.997 .

Ｒ函數：Gamma(shape, rate)

指數分布 (Exponential Distribution)

P(X=x)=1βe−x/β=λe−λxP(X=x) = \frac{1}{\beta} e^{-x/\beta} = \lambda {e}^{- \lambda x}P(X=x)=β1​e−x/β=λe−λx

  • 範圍：x=0,…,k （取出 x 個白球）
  • 意義：白球有 m 個，黑球 n 個，取出 k 個球，其中有 x 個白球的機率; (取後不放回)
  • R 函數： [exp(rate);rate=1/β=λ exp(rate) ; rate = 1/\beta = \lambda exp(rate);rate=1/β=λ]]
    • http://stat.ethz.ch/R-manual/R-patched/library/stats/html/Hypergeometric.html

特性

  1. E(X)=k(mm+n)E(X) = k (\frac{m}{m+n})E(X)=k(m+nm​)
  2. Var(X)=k(nm+n)(mm+n)(m+n−km+n−1)Var(X) = k (\frac{n}{m+n}) (\frac{m}{m+n}) (\frac{m+n-k}{m+n-1})Var(X)=k(m+nn​)(m+nm​)(m+n−1m+n−k​)

動差生成函數：mx(t)=(1−βt)−1m_x(t) = (1-\beta t)^-1mx​(t)=(1−βt)−1

卡方分布 (Chi-Square distribution)

公式： 12γ/2Γ(γ/2)xγ/2−1e−x/2\frac{1}{2^{\gamma/2}\Gamma(\gamma/2)}\,x^{\gamma/2 - 1} e^{-x/2}2γ/2Γ(γ/2)1​xγ/2−1e−x/2

R 的公式： chisq(df=n>0)=fn(x)=1(2n/2Γ(n/2)xn/2−1e−x/2chisq(df=n > 0) = f_n(x) = \frac{1}{(2^{n/2} \Gamma(n/2)} x^{n/2-1} e^{-x/2}chisq(df=n>0)=fn​(x)=(2n/2Γ(n/2)1​xn/2−1e−x/2

Ｒ函數：chisq(df=n, ncp=λ)

  • 公式： f(x) = exp(-λ/2) SUM_{r=0}^∞ ((λ/2)^r / r!) dchisq(x, df + 2r)
  • 網址：http://stat.ethz.ch/R-manual/R-patched/library/stats/html/Chisquare.html

特性

卡方分配主要用途是在「利用樣本推斷母體變異數」時使用的。

卡方分配是伽瑪分布的特例，只要將 Gamma 分配中設定 b=2,a=γ/2b=2, a=\gamma/2b=2,a=γ/2 就是卡方分布了。

stopifnot(all.equal(p00, exp(-lam/2)),
          all.equal(p.0, exp(-lam/2)))

T 分布 (T distribution)

T 分配：Z 為標準常態分布，Xγ2X_\gamma^2Xγ2​ 為自由度 γ\gammaγ 的卡方分布。

公式： T=ZXγ2/γT = \frac{Z}{\sqrt{X_\gamma^2/\gamma}}T=Xγ2​/γ​Z​

T 分布公式： f(t)=Γ(γ+1)/2Γ(γ/2)πγ(1+t2γ)−(γ+1)/2f(t) = \frac{\Gamma{(\gamma+1)}/2}{\Gamma(\gamma/2) \sqrt{\pi \gamma}} (1+\frac{t^2}{\gamma})^{-(\gamma+1)/2}f(t)=Γ(γ/2)πγ​Γ(γ+1)/2​(1+γt2​)−(γ+1)/2 ; 範圍：−∞<t<∞-\infty < t < \infty−∞<t<∞

Ｒ函數：t(df, Del)

  • 公式：T(df=n) := f(x) = Γ((n+1)/2) / (√(n π) Γ(n/2)) (1 + x^2/n)^-((n+1)/2)
  • 擴充：T(df, Del) := (U + Del) / √(V/df) ; U ~ N(0,1) , V ~ χ^2(df) 且 (U,V) 獨立。
  • 網址：http://stat.ethz.ch/R-manual/R-patched/library/stats/html/TDist.html
