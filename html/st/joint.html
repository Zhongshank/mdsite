表示符號： PXY(x,y)=P[X=x,Y=y]P_{XY}(x,y) = P[X=x, Y=y]PXY​(x,y)=P[X=x,Y=y]

必要條件：

  1. PXY(x,y)≥0P_{XY}(x,y) \ge 0PXY​(x,y)≥0 ;

  2. ∑∀x∑∀yPXY(x,y)=1\sum{\forall x} \sum{\forall y} P_{XY}(x,y) = 1∑∀x​∑∀y​PXY​(x,y)=1 ;

定義 2：連續聯合密度函數

表示符號： PXY(x,y)P_{XY}(x,y)PXY​(x,y) (範圍：−∞<x<∞;−∞<y<∞;-\infty < x < \infty;-\infty < y < \infty;−∞<x<∞;−∞<y<∞;)

  1. PXY(x,y)≥0P_{XY}(x,y) \ge 0PXY​(x,y)≥0 ;

  2. ∫−∞∞∫−∞∞PXY(x,y)dydx=1\int{-\infty}^{\infty} \int{-\infty}^{\infty} P_{XY}(x,y) dy dx = 1∫−∞∞​∫−∞∞​PXY​(x,y)dydx=1 ;

聯合分配的期望值

  1. 離散的情況： E[H(X,Y)]=∑∀x∑∀yH(x,y)PXY(x,y)E[H(X,Y)]=\sum{\forall x} \sum{\forall y} H(x,y) P_{XY}(x,y)E[H(X,Y)]=∑∀x​∑∀y​H(x,y)PXY​(x,y)

  2. 連續的情況： E[H(X,Y)]=∫−∞∞∫−∞∞H(x,y)PXY(x,y)dydxE[H(X,Y)]=\int{-\infty}^{\infty} \int{-\infty}^{\infty} H(x,y) P_{XY}(x,y) dy dxE[H(X,Y)]=∫−∞∞​∫−∞∞​H(x,y)PXY​(x,y)dydx

定義：聯合分配中單一變數的期望值

  1. 離散： E[X]=∑∀x∑∀yxPXY(x,y)=∑∀xxPX(x)E[X] = \sum{\forall x} \sum{\forall y} x P{XY}(x,y) = \sum{\forall x} x P_{X}(x) E[X]=∑∀x​∑∀y​xPXY​(x,y)=∑∀x​xPX​(x)

  2. 離散： E[Y]=∑∀y∑∀xyPXY(x,y)=∑∀y∑∀yyPY(y)E[Y] = \sum{\forall y} \sum{\forall x} y P{XY}(x,y) = \sum{\forall y} \sum{\forall y} y P{Y}(y)E[Y]=∑∀y​∑∀x​yPXY​(x,y)=∑∀y​∑∀y​yPY​(y)

  3. 連續： E[X]=∫−∞∞∫−∞∞xPXY(x,y)dydx=∫−∞∞xPX(x)dxE[X] = \int{-\infty}^{\infty} \int{-\infty}^{\infty} x P{XY}(x,y) dy dx= \int{-\infty}^{\infty} x P_{X}(x) dxE[X]=∫−∞∞​∫−∞∞​xPXY​(x,y)dydx=∫−∞∞​xPX​(x)dx

  4. 連續： E[Y]=∫−∞∞∫−∞∞yPXY(x,y)dydx=∫−∞∞yPY(y)dyE[Y] = \int{-\infty}^{\infty} \int{-\infty}^{\infty} y P{XY}(x,y) dy dx = \int{-\infty}^{\infty} y P_{Y}(y) dyE[Y]=∫−∞∞​∫−∞∞​yPXY​(x,y)dydx=∫−∞∞​yPY​(y)dy

共變異數 (Covariance, 協方差)

定義：共變異數 Cov(X,Y)

σXY=Cov(X,Y)=E[(X−μX)(Y−μY)]\sigma_{XY} = Cov(X, Y) = E[(X - \mu_X)(Y - \mu_Y)]σXY​=Cov(X,Y)=E[(X−μX​)(Y−μY​)]

定理：共變異數與期望值之關係

Cov(X,Y)=E[XY]−E[X]E[Y]Cov(X, Y) = E[X Y] - E[X] E[Y]Cov(X,Y)=E[XY]−E[X]E[Y]

定理：相互獨立時的期望值

如果 X, Y 相互獨立 ，則 E[X Y] = E[X] E[Y] 。

多變數聯合分布的情況

聯合分布與條件機率

定義：如果 X,Y 滿足下列條件，則稱 X, Y 兩者之間獨立：

PXY(x,y)=PX(x)PY(y)P{XY}(x,y) = P{X}(x) P_{Y}(y)PXY​(x,y)=PX​(x)PY​(y)

多個變數的貝氏定理

  1. P(A,B∣C)=P(C∣A,B)∗P(A,B)P(C)P(A,B|C) = P(C|A,B) * \frac{P(A,B)}{P(C)}P(A,B∣C)=P(C∣A,B)∗P(C)P(A,B)​ ;

  2. P(A∣B,C)=P(B,C∣A)∗P(A)P(B,C)P(A|B,C) = P(B,C|A) * \frac{P(A)}{P(B,C)}P(A∣B,C)=P(B,C∣A)∗P(B,C)P(A)​ ;

  3. P(A,B∣C,D)=P(C,D∣A,B)∗P(A,B)P(C,D)P(A,B|C,D) = P(C,D|A,B) * \frac{P(A,B)}{P(C,D)}P(A,B∣C,D)=P(C,D∣A,B)∗P(C,D)P(A,B)​ ;

其他情況可以類推，只要能正確改寫 A , B 為任何隨機變數序列都行。

條件獨立與貝氏定理

假如 A 與 B 在給定 C 的情況下條件獨立，那麼以下算式成立：

P(A,B∣C)=P(A∣C)∗P(B∣C)=P(C∣A)∗P(C∣B)∗P(A)P(B)P(C)2P(A,B|C) = P(A|C) P(B|C) = P(C|A) P(C|B)*\frac{P(A) P(B)}{P(C)^2}P(A,B∣C)=P(A∣C)∗P(B∣C)=P(C∣A)∗P(C∣B)∗P(C)2P(A)P(B)​

結語

兩個位於樣本空間 S 的聯合隨機行為，會導致樣本空間變成其迪卡兒乘積 S×S，其樣本空間大小變成 ∣S∣2|S|^2∣S∣2 。

而 k 個位於樣本空間 S 的聯合隨機行為，會導致樣本空間變成其迪卡兒乘積 S×S×...×S ，其樣本空間大小變成 ∣S∣n|S|^n∣S∣n 。

如果兩個位於不同樣本空間 (SX,SY)(S_X, S_Y)(SX​,SY​) 的聯合隨機行為，則會導致樣本空間變成 SX×SYS_X \times S_YSX​×SY​ ，其樣本空間大小變為 ∣SX∣∣SY∣|S_X| |S_Y|∣SX​∣∣SY​∣ 。

此時 X, Y 的機率密度函數將會採用以下的「邊際機率密度函數」之算法，以便將聯合樣本空間 (SX,SY)(S_X, S_Y)(SX​,SY​) 中的機率與單一樣本空間 SXS_XSX​ 或 SYS_YSY​ 中的機率關聯起來。

P(X=x)=P(x,∗)=∑y∈SYP(x,y)P(X=x) = P(x, *) = \sum_{y \in S_Y} P(x,y)P(X=x)=P(x,∗)=∑y∈SY​​P(x,y)

P(Y=y)=P(∗,y)=∑x∈SXP(x,y)P(Y=y) = P(*, y) = \sum_{x \in S_X} P(x,y)P(Y=y)=P(∗,y)=∑x∈SX​​P(x,y)

最後我們必須強調的是，樣本空間的選擇並沒有一定的標準，您可以視問題的需要來定義樣本空間，通常我們會盡量利用獨立的特性，讓樣本空間越小越好，否則將會很難計算。
